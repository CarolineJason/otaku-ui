## Select 下拉框

::: demo

简单的评分

```tsx
function Demo () {
  return (
    <Select>
      {
        new Array(20).fill().map((_, index) => {
          return <SelectOptions key={index}>第{index + 1}个选项</SelectOptions>
        })
      }
    </Select>
  )
}

ReactDOM.render(<Demo/>, container)

```
:::
